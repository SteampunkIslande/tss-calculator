# TSS Calculator

This simple, standalone HTML web page allows one to easily compute Thalassemia Severity Score (TSS). Just download [the english version](calculator-en.html) on your computer, no need to install anything, and open it in the browser (only tested on Firefox but it should work on any browser).

Select the genotypes that apply to your sample, and get the corresponding TSS.

The score is computed according to this 2015 paper (open access, [doi:10.3324/haematol.2014.113886](https://haematologica.org/article/view/7336)).

# Calculateur de score de sévérité de Thalassémie

Cette simple page HTML, standalone, vous permet de calculer facilement le score de sévérité de Thalassémie (TSS en anglais). Téléchargez [la version en français](calculator-fr.html) sur votre ordinateur, il n'y a rien à installer, et ouvrez le fichier dans votre navigateur web (testé sur Firefox uniquement mais devrait fonctionner sur n'importe quel navigateur).

Sélectionnez les génotypes qui correspondent à votre échantillon, et obtenez le TSS associé.

Le score est calculé selon la méthode décrite dans cet article de 2015 (open access, [doi:10.3324/haematol.2014.113886](https://haematologica.org/article/view/7336)).
